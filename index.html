What's inside

Built for how retail actually works

Every feature exists to solve a real problem a garment shop owner faces every single day.

GST-Compliant Billing Engine
Auto-computed CGST + SGST split based on per-product HSN code and rate. Sequential atomic bill numbers (JB-1001+). A4 invoice and 80mm thermal print templates with QR code verification. One-click WhatsApp share. Tally XML export for accountant handoff.
Live
Variant Inventory Matrix
Track stock per Color × Size variant using MongoDB nested Maps. Auto-deducts on sale, auto-restores on cancellation. DB-level low stock filters via aggregation pipelines.
Live
Sales Report & P&L
Daily, weekly, or monthly granularity. Revenue vs Expenses vs Profit chart — profit bars turn red on loss months. Top 10 products, payment mix, status breakdown, and expense category breakdown. Export the full timeline as CSV.

Powered by a single GET /dashboard/report endpoint that runs 6 parallel MongoDB aggregations.
Expense-aware
Customer Ledger
Per-customer transaction history with lifetime value, outstanding dues, and last purchase — in one aggregation, no N+1.
Live
Security & Audit Trail
HttpOnly refresh tokens, 8 granular permissions, PII masking, and a 90-day TTL audit log for every write operation. Rate-limited at 200 req/15min.
